# Taller Patrón Observer - Configuration Manager

## 📋 Información General

- **Asignatura:** Patrones de Diseño de Software
- **Universidad:** Universidad de La Sabana
- **Patrón implementado:** Observer (Patrón de Comportamiento)
- **Lenguaje:** Java
- **Build Tool:** Gradle

---

## 🎯 Objetivo del Taller

Implementar el **Patrón Observer** para crear un sistema de configuración que notifique automáticamente a los objetos interesados cuando ocurren cambios en la configuración del sistema, eliminando la necesidad de consultas constantes (polling).

### Intención del Patrón
> Define una dependencia de uno-a-muchos entre objetos, de modo que cuando un objeto cambia su estado, todos sus dependientes son notificados y actualizados automáticamente.
